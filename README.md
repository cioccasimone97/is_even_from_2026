# is_even_from_2026 🚀

The most advanced, deterministic, and absolutely massive parity-checking library of 2026. Why use the `% 2` operator when you can have millions of lines of dynamically loaded `if` statements?

## 🛠 How to contribute to the madness (TODO)

If you want to help expand the database of supported numbers, follow these strict steps:

### 1. Clone the branch

Start by downloading this masterpiece onto your local machine.

### 1.5 Check disk space ⚠️

WARNING: Each generation batch adds ~1 GB of files. Make sure you have enough space on your SSD before proceeding. We take no responsibility for exploded hard drives or completely filled partitions.

### 2. Run the generator

Execute the `generator.py` script from the root of the project. The script will automatically detect the last generated number and produce the next 2000 files.

### 3. Wait

The generator is multithreaded, but we’re still writing millions of lines of code. This is the perfect time to grab a coffee, watch a movie, or reflect on your life choices.

### 4. Commit your progress

Once the generation is complete, commit the new files. The commit message must be specific to indicate the latest milestone reached:
"added [LAST_FILE_NUMBER]"
