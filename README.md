# is_even_from_2026 🚀

La libreria di controllo parità più avanzata, deterministica e mastodontica del 2026. Perché usare l'operatore `% 2` quando puoi avere milioni di righe di `if` caricate dinamicamente?

## 🛠 Come contribuire alla follia (TODO)
Se vuoi aiutare a espandere il database dei numeri supportati, segui questi passaggi rigorosi:

### 1. Clonare il branch
Inizia scaricando l'opera d'arte sul tuo computer locale

### 1.5 Verificare lo spazio sul PC ⚠️
ATTENZIONE: Ogni batch di generazione aggiunge ~1 GB di file. Assicurati di avere spazio a sufficienza sul tuo SSD prima di procedere. Non ci assumiamo responsabilità per dischi rigidi esplosi o partizioni saturate.

### 2. Eseguire il generatore
Lancia lo script generator.py dalla root del progetto. Lo script rileverà automaticamente l'ultimo numero generato e produrrà i successivi 2000 file.

### 3. Attendere
Il generatore è multithread, ma stiamo pur sempre scrivendo milioni di righe di codice. È il momento perfetto per farti un caffè, guardare un film o riflettere sulle tue scelte di vita.

### 4. Committare i progressi
Una volta terminata la generazione, esegui il commit dei nuovi file. Il messaggio di commit deve essere specifico per indicare l'ultimo traguardo raggiunto: "added [ULTIMO_NUMERO_FILE]"
